# lampp-auto
Enjoy and have fun

- Install
```
sudo apt update
sudo apt install git
sudo git clone https://github.com/b0rnt0ber00t/lampp-auto.git && cd lampp-auto
sudo chmod +x install.sh
sudo ./install.sh
```
